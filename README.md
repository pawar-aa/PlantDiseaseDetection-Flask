# PlantAI: Disease Prediction Web App

This is a web application for predicting diseases based on uploaded images.
<br>

## Features
- Drag and drop or click to upload images.
- Predicts diseases based on uploaded images.
- Displays weather information based on the user's location.
- Open-source project developed by Aashay Pawar under the YourStory Hackathon 2024.
<br>
